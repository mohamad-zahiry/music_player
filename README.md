# music_player
A simple music player with python and **tkinter** and **python-vlc**

---

## Supported formats
It supports most known formats such as `mp3`, `mp4`, `wav`, `flac`, `aac`, `ac3`, `m4a`, `aiff`, `ogg`, `adts`, `opus`, `ts`


---

## Features
- volume control
- amplifier
- bitrate control
- fast forward|backward
- add bulk music to play list
